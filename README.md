# MOKO_ExScript

Репозиторий, содержащий тестовые и демонстрационные Python-скрипты для работы с платформой MOKO SE.

## Описание

Этот проект, по-видимому, используется для тестирования различных функций библиотеки `MOKO.py` и демонстрации ее возможностей. Скрипты показывают, как взаимодействовать с различными компонентами MOKO SE, такими как:

- **AutoIt:** Автоматизация взаимодействия с GUI-приложениями.
- **WordReport:** Создание отчетов в формате MS Word.

Проект также содержит файл `qodana.yaml`, что указывает на использование статического анализатора кода Qodana для обеспечения качества кода.

## Структура проекта

- **Tests/**: Папка с тестовыми скриптами.
  - **Autoit/**: Тесты для интеграции с AutoIt.
  - **WordReport/**: Тесты для генерации отчетов Word.
- **MOKO.py**: Копия основной библиотеки для взаимодействия с MOKO SE по HTTP.
- **qodana.yaml**: Файл конфигурации для статического анализатора Qodana.

## Ключевые технологии

- Python
- Qodana
